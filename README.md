# Crime Data Analysis and Prediction

Data analysis and machine learning project on crime trends using real-world data.

---

## 📊 Project Overview
This project focuses on analyzing crime data to identify patterns and trends using data visualization techniques and applying machine learning models to predict crime occurrences.

---

## 🔍 Features
- Data cleaning and preprocessing using Pandas & NumPy  
- Exploratory Data Analysis (EDA)  
- Data visualization using Matplotlib & Seaborn  
- Analysis of crime trends across time and location  
- Victim-based analysis (age and gender)  
- Handling missing and invalid data  
- Machine learning model building and comparison  

---

## 📈 Key Insights
- Crime rates vary across different areas and time periods  
- Certain age groups are more affected by crimes  
- Data quality (cleaning and preprocessing) significantly impacts analysis  
- Proper data aggregation improves model performance  

---

## 🤖 Machine Learning
- Models Used:
  - Linear Regression  
  - Random Forest  
  - XGBoost  
  - LightGBM  
  - CatBoost  

- Best Model: **CatBoost Regressor**  
- Achieved a strong R² score (~0.50)  
- Selected based on best performance and generalization
